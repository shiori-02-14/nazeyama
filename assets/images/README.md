# 画像について

このサイトは**画像が無くても表示されます**（ヒーローやネッコはコード内のSVGイラストで描いています）。
差し替えたい場合は、このフォルダに以下のファイル名で画像を置いてください（すべて任意）。

| ファイル名 | 用途 | 推奨サイズ |
| --- | --- | --- |
| `hero.png` | ヒーローのキービジュアル（黒板＋白衣の物理学者 など） | 横長 |
| `neko.png` | 物理屋のネッコ | 正方形 |
| `yama.png` | 砂山の少年（Xアイコン） | 正方形 |
| `ogp.png` | SNSシェア用サムネ | 1200×630px |

書籍の表紙画像を使いたい場合も、ここに置いて `content/books.yaml` から参照する形に拡張できます。

> メモ: X アイコン（砂山の少年）は「いらすとん」(https://www.irasuton.com/) の無料素材です。
> 本人が継続使用中のため当面そのまま使用します（商用可・クレジット不要）。
